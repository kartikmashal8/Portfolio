Not used by default — every icon in this site (GitHub, LinkedIn,
Instagram, mail, phone, download, etc.) is written as inline SVG
directly in index.html, so there are no external icon files to manage
and nothing to break if a file goes missing.

If you'd rather use icon image files instead, drop them here and swap
the relevant <svg>...</svg> blocks in index.html for <img> tags
pointing at assets/icons/yourfile.svg.
